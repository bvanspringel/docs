title: Introduction
---

This guide walks you through the key steps for powering up, configuring, and connecting your Power Platform controller so it integrates seamlessly into your local 
network. You’ll learn how to start the controller, adjust its network settings, update the default credentials, and install the applications you need. The guide also 
covers setting up remote support access to keep your system connected and manageable.

Each section is designed to be clear and practical, ensuring a smooth setup experience from start to finish. Whether you’re working with the standalone or panel 
version of the Power Platform controller, you’ll find all the essential instructions here to get your system up and running with confidence.
