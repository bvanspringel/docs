### Change Admin Password

<Note>
It is highly recommended to change the default password for the admin user provided by DEP. Doing so ensures that the controller 
is more secure with your own selected password.
</Note>

To change the admin user password, follow these steps:

<Steps>
  <Step title="Login to the WBM">
    <Card
      title="Login to the WBM"
      icon="lock-open"
      href="/installation/settings/login"
      horizontal
    >
      Login to the Web-Based Management (WBM) interface of the Power Platform controller.
    </Card>

  </Step>
  <Step title="Navigate to the Security Tab">
    Navigate to the **Security** tab in the left navigation plane and select **User Authentication**.
  </Step>
  <Step title="Change Password">
    For the admin user, click on **Set Password**.
    Type in the new password for the admin user and confirm.
    <img
      src="/images/wbm_security_user_auth.png"
      alt="Change Credentials"
      style={{ width: '400px', margin: 'auto' }}
    />
  </Step>
  <Step title="Save Password">
    Press the **Save** button to save the new password.
    <img
      src="/images/wbm_save_password.png"
      alt="Save Password"
      style={{ width: '400px', margin: 'auto' }}
    />
  </Step>
</Steps>
