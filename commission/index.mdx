title: Introduction
---

This section provides detailed instructions on verifying communication between the controller and devices within the 
microgrid, an essential step in the commissioning process. The methods covered include using MQTT Explorer and a custom 
NATS application to monitor and retrieve device states.

Before proceeding, please ensure you have completed the necessary steps outlined in the previous sections, including 
powering up the unit, modifying the configuration files, updating the device files, and successfully rebooting the 
controller. These prerequisites are essential for following the instructions in this document.
